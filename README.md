# AI Movie Recommendation System

This project is a beginner machine learning system that recommends movies based on genre similarity.

live site:
https://ai-movie-recommendation-dashboard.onrender.com

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Machine Learning

## How It Works
The system converts movie genres into numeric vectors using CountVectorizer and calculates similarity using cosine similarity.

## Features
- Recommends movies based on genre similarity
- Handles invalid movie input
- Displays available movies
